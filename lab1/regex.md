# 正则表达式

思考题

```\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*``` 正则表达式匹配的字符串的含义是什么？

先匹配任意 (>=1) 个数字，然后可以有或没有若干个 (>=0) -+. 字符，然后是一个 @ ，然后又是若干个 (>=1) 然后有或没有 (>=0) "一个 -.和多个 (>=1) 数字" 然后一个 . 然后多个 (>=1) 数字，然后是多个 (>=0) "-或. 和多个 (>=1) 数字"。

匹配 HTML 注释：编写一个正则表达式，可以匹配 HTML 中的注释，例如 ```<!-- This is a comment -->```。

```regex
<!--.*?-->
```
